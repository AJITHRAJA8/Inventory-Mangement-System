# 📖 About

The **Inventory Management System** is a web application developed using **Python Flask** and **Microsoft SQL Server**. It helps businesses manage products, categories, suppliers, and inventory efficiently.

This project was created to strengthen my knowledge of **Python Full Stack Development**, **SQL Server**, **Stored Procedures**, **Flask Blueprints**, and **Database Management**.

# ⚙️ Installation

```bash
# Clone the repository

git clone https://github.com/AJITHRAJA8/Inventory-Mangement-System.git

# Go to project folder

cd Inventory-Mangement-System

# Create virtual environment

python -m venv venv

# Activate environment

venv\Scripts\activate

# Install dependencies

pip install -r requirements.txt

# Run project

python app.py
```
